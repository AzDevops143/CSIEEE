Academic Context & Background

SNOOPFINGER is a cross-modality side-channel attack targeting head-mounted displays (HMDs) in VR and AR environments. When users type on virtual keyboards, subtle head movement occurs as they visually confirm keys. Background applications can continuously access head orientation sensors without any security permissions, collecting high-frequency gaze coordinates.

To mitigate this, we present the Context-Aware Sensor Obfuscation Middleware (CASOM). By monitoring typing states, it detects keyboard interactions. When active, it adds Laplacian noise ($\mu = 0, b$) to the coordinate stream sent to background threads, scattering the gaze cluster points:

x_{obf} = x_{raw} + Laplace(0, b)

The attacker relies on spatial clustering (finding where points linger near key centers) to reconstruct the keystrokes. As shown above, adding noise scatters the centroids outside the key mapping radius, dropping the attacker's reconstruction success to 0% while preserving foreground usability.
